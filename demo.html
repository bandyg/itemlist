🚀 启动说明

由于 Node.js 版本兼容性问题，建议使用以下方法之一启动开发服务器：

  1. 使用 Node.js 16 或更低版本：
    nvm use 16 (如果安装了 nvm)
    ng serve
  2. 或者构建生产版本：
    ng build --prod

📝 可用命令

  • ng serve - 启动开发服务器
  • ng build - 构建项目
  • ng test - 运行单元测试
  • ng e2e - 运行端到端测试
  • ng generate component [name] - 生成新组件
